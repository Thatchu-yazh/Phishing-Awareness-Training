# PHISHING AWARENESS TRAINING

## 📌 Project Overview

This project was developed as part of the CodeAlpha Cyber Security Internship program. The main objective of this project is to educate users about phishing attacks, email security threats, fake websites, and social engineering techniques used by cybercriminals.

Phishing is one of the most common cyber attacks used to steal sensitive information such as usernames, passwords, banking credentials, OTPs, and personal data. Through this awareness training presentation, users can understand how phishing attacks work and learn best practices to stay safe online.

---

# 🎯 Objectives

- To create awareness about phishing attacks
- To educate users on identifying fake emails and websites
- To explain different types of phishing techniques
- To improve cyber security awareness among users
- To provide prevention and safety measures against phishing attacks

---

# 🛡 Topics Covered

## 1. Introduction to Phishing
- Definition of phishing
- Importance of cyber awareness
- Real-world phishing examples

## 2. Types of Phishing Attacks
- Email phishing
- Spear phishing
- Smishing (SMS phishing)
- Vishing (Voice phishing)

## 3. Email Security Awareness
- Identifying suspicious emails
- Detecting fake sender addresses
- Recognizing malicious attachments
- Avoiding suspicious links

## 4. Fake Website Detection
- Checking HTTPS security
- Identifying fake URLs
- Detecting cloned login pages
- Website verification techniques

## 5. Social Engineering Attacks
- Psychological manipulation techniques
- Fear and urgency tactics
- Fake technical support scams
- Identity impersonation attacks

## 6. Cyber Safety Practices
- Strong password usage
- Two-factor authentication
- Antivirus protection
- Software updates
- Safe browsing practices
